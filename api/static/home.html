How It Works

Four steps from discovery to cryptographic proof.

01

Discover

Any agent finds you via /.well-known/x402 or the on-chain Soroban registry.

02

Pay

Send USDC via x402 v2 or fallback XLM — payment is the authorization.

03

Execute

Your job runs in a sandboxed Docker container (no network by default, 256MB RAM, read-only FS).

04

Verify

Receive a result signed with Ed25519 — cryptographically provable, tamper-proof.

Supported Protocols

Standards that make agent commerce interoperable.

x402

Payment layer. Client pays USDC, retries with proof. Facilitators: Coinbase (x402.org) and OpenZeppelin.

A2A

Agent-to-Agent communication. Full SendMessage, GetTask, ListTasks, CancelTask over HTTP+JSON and JSON-RPC.

EIP-8004

On-chain agent registration. Soroban identity with stellar:{NETWORK}:{CONTRACT_ID} format.
